# projects/ — active work

One file per project. Each file holds everything about that project:
goal, research, drafts, decisions, TODOs. Files start with a one-line
goal and a `Status:` line (`active` / `stalled` / `shipped`).

When a project ships, run `/harvest <project>` — its lessons go to
`wiki/`, its artifacts and archived file go to `output/`.
